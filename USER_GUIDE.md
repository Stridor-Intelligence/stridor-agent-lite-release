# Stridor Agent System Lite User Guide

This guide is under development. It will be expanded as the macOS development beta stabilises.

## 1. Download

Go to the [Releases page](https://github.com/Stridor-Intelligence/stridor-agent-lite-release/releases) and download the correct macOS build for your Mac.

- Apple Silicon: M1, M2, M3, M4 and newer Macs
- Intel: older Intel-based Macs

Each release includes a checksum file so you can verify the download.

## 2. Install

Open the downloaded `.dmg` file and drag Stridor Agent System Lite into your Applications folder.

During the development beta, macOS may warn that the app is from an unidentified developer. This is expected while packaging and signing are still being improved.

## 3. First Launch

Open Stridor Agent System Lite from Applications.

The app creates a local Stridor workspace on first launch. Existing user workspaces should not be overwritten or repaired by the app.

## 4. Models

We suggest using the free OpenZen model path when available.

You can also bring your own model or provider if you prefer.

## 5. Updates

Check the [Releases page](https://github.com/Stridor-Intelligence/stridor-agent-lite-release/releases) for new builds, version tags, checksums, and update notes.

## 6. Windows

Windows is planned but not available yet. A waiting list will be published from the Stridor website when it opens.

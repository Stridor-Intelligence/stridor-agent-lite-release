# Stridor Agent Lite

**Your companion for the DeRose Method — strength, flexibility and meditation, from your desktop.**

Stridor Agent Lite is a desktop app for students and practitioners of the DeRose Method. Access your practice tools, track your progress, and stay connected — all from your Mac or PC.

---

## What This Repo Is

This is the **official release distribution point** for Stridor Agent Lite. Only compiled installers are published here — no source code. Download the version for your platform and you're ready to go.

Built with [Electron](https://www.electronjs.org/) and packaged with [electron-builder](https://www.electron.build/).

## System Requirements

- **macOS:** 12 (Monterey) or later
- **Windows:** Windows 10 (64-bit) or later
- **Storage:** ~500 MB free space

---

## Installation

### macOS

1. Download the `.dmg` file for your architecture (Apple Silicon or Intel).
2. Open the disk image and drag **Stridor Agent Lite** into your **Applications** folder.
3. Launch from Applications (or your Dock).

### Windows

1. Download the `.exe` installer.
2. Run the installer and follow the on-screen prompts.
3. Launch from the Start Menu or Desktop shortcut.

---

## Updating

Stridor Agent Lite checks for updates automatically on launch. When a new version is available, you'll be prompted to download and install it — no need to visit this repo again unless you want the latest installer directly.
